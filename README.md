# Парсер контактов цветочных магазинов

Скрипт для сбора контактных данных (телефон, email) цветочных магазинов из поисковой выдачи Яндекса.

> Задание выполнено в рамках летней практики в компании **МТС**.

## Что делает скрипт

1. Ищет сайты цветочных магазинов в Яндексе по нескольким городам Беларуси.
2. Заходит на каждый найденный сайт и извлекает телефон и email (при необходимости — заглядывает в раздел «Контакты»).
3. Сохраняет результаты в файл `auto_flower_contacts.csv`.

## Установка

```bash
pip install playwright
playwright install chromium
```

## Запуск

```bash
python main.py
```

При первом запуске откроется окно браузера — при появлении капчи от Яндекса нужно решить её вручную, скрипт продолжит работу автоматически.

## Результат

CSV-файл `auto_flower_contacts.csv` со столбцами:

| Сайт | Телефон | Email |
|------|---------|-------|
